# Atelier — Angular

A workshop toolkit for hands-on AI development training. 22 LLM-optimized components, zero runtime dependencies, and a built-in MCP server so your AI assistant can look up exact prop names instead of guessing them.

**Package:** `@atelier-ui/angular` &nbsp;·&nbsp; **Selectors:** `llm-button`, `llm-dialog`, … &nbsp;·&nbsp; **Style:** standalone components + CSS custom properties

---

## Quick start

```bash
npm install @atelier-ui/angular
```

```ts
// angular.json → styles
"styles": ["node_modules/@atelier-ui/angular/styles/tokens.css"]
```

```ts
import { LlmButtonComponent } from '@atelier-ui/angular';

@Component({
  imports: [LlmButtonComponent],
  template: `<llm-button variant="primary">Save</llm-button>`,
})
export class MyComponent {}
```

## MCP server

The MCP server is hosted and pre-configured in workshop workspaces. Add it manually to any project:

```json
{
  "mcpServers": {
    "storybook-angular": {
      "type": "http",
      "url": "https://atelier.pieper.io/storybook-angular/mcp"
    }
  }
}
```

Once connected, your AI assistant can call `get_component_docs`, `search_components`, `get_stories`, and `get_theming_guide` — getting exact selectors, inputs, outputs, and defaults without hallucinating the API.
