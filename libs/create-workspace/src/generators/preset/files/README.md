# Atelier UI Workshop

This workspace was scaffolded with the Atelier UI workshop preset.

## Getting started

```bash
npm install
npx nx serve workshop
```

## Components

`@atelier-ui/angular` is installed and ready to use. Import components directly:

```typescript
import { LlmButtonComponent } from '@atelier-ui/angular';
```

## MCP

The Claude Code MCP server is pre-configured in `.claude/settings.json`.
Your AI assistant can look up component props and usage examples directly
from the Atelier UI Storybook.

## Storybook

Browse all components at https://atelier-ui.netlify.app/storybook-angular/
