# Colors

All color tokens use the `--ui-color-*` prefix and automatically switch between light and dark themes via the `data-theme="dark"` attribute.

The current palette is **Direction C** ("workshop signature"): Conciso navy spine + Pieper-organic moss accent on a warm-bone canvas. Light-mode-first; dark mode is a switch. All foreground/background pairs are verified to meet WCAG 2.2 AA.
