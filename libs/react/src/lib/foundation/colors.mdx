# Colors

All color tokens use the `--ui-color-*` prefix and automatically switch between light and dark themes via the `data-theme="dark"` attribute.

## Dark Theme Overrides

When `data-theme="dark"` is applied these tokens resolve to their dark-mode equivalents automatically.

| Token | Light | Dark |
|---|---|---|
| `--ui-color-primary` | `#00bebe` | `#00d0d0` |
| `--ui-color-surface` | `#ffffff` | `#1e1e2e` |
| `--ui-color-surface-sunken` | `#f5f5f5` | `#18181f` |
| `--ui-color-text` | `#0f172a` | `#f1f5f9` |
| `--ui-color-border` | `#e5e7eb` | `#2e2e3e` |
| `--ui-color-danger` | `#ef4444` | `#f87171` |
