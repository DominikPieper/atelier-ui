# Atelier — React

A workshop toolkit for hands-on AI development training. 22 LLM-optimized components, zero runtime dependencies, and a built-in MCP server so your AI assistant can look up exact prop names instead of guessing them.

**Package:** `@atelier-ui/react` &nbsp;·&nbsp; **Elements:** `LlmButton`, `LlmDialog`, … &nbsp;·&nbsp; **Style:** CSS custom properties

---

## Quick start

```bash
npm install @atelier-ui/react
```

```css
/* index.css or global stylesheet */
@import '@atelier-ui/react/styles/tokens.css';
```

```tsx
import { LlmButton } from '@atelier-ui/react';

export function MyComponent() {
  return <LlmButton variant="primary">Save</LlmButton>;
}
```

## What's inside

| Category | Components |
|---|---|
| **Inputs** | Button, Input, Textarea, Checkbox, Toggle, Select, RadioGroup |
| **Display** | Badge, Card, Avatar, Skeleton, Progress |
| **Navigation** | Breadcrumbs, Tabs, Pagination, Menu |
| **Overlay** | Dialog, Drawer, Tooltip, Toast |

## MCP server

The MCP server is hosted and pre-configured in workshop workspaces. Add it manually to any project:

```json
{
  "mcpServers": {
    "storybook-react": {
      "type": "http",
      "url": "https://atelier.pieper.io/storybook-react/mcp"
    }
  }
}
```

Once connected, your AI assistant can call `get_component_docs`, `search_components`, `get_stories`, and `get_theming_guide` — getting exact element names, prop types, and defaults without hallucinating the API.
