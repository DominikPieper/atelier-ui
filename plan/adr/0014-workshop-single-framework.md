# ADR-0014: Workshop covers ONE framework; the 3-framework rig is prep infra

## Status

Accepted. Confidence: **documented** (quoted from the repo).

## Context

A user correction resolved a misread that the three-framework setup was "overcomplicated" curriculum that should be cut. The three-framework rig is preparation infrastructure, not what learners are taught.

## Decision

each cohort picks one framework; cross-framework sync is invisible to learners. Why: corrects the "overcomplicated / cut a framework" misread — the rig exists so any chosen framework is ready, not as curriculum. Relates to ADR-0005.

## Consequences

- Each cohort learns exactly one framework; the other two stay out of view.
- The 3-framework rig (ADR-0005) is justified as readiness infra, not teaching load.
- Cross-framework sync stays invisible to learners.
